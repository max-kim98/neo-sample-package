# neo-sample-package

Frontend + backend package test project following the package guide and neo-cat style layout.

## Project Name Rule

Package name is derived from the git origin repository name.

- Shell: `./scripts/project-name.sh`
- PowerShell: `./scripts/project-name.ps1`

Example: `https://github.com/max-kim98/neo-sample-package.git` -> `neo-sample-package`
